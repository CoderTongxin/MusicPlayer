React-music-playera
===

A music player build with react, react-router,webpack，ES6

音乐播放器带有播放下一曲，上一曲，暂停，调整进度和音量，查看音乐列表，切换播放模式等功能

下载代码

    git clone https://github.com/CoderTongxin/MusicPlayer.git
安装依赖

    npm install
项目启动

    npm start
具体可看package.json配置

        npm run dev
编译打包

        npm run build
