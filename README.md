React-music-player
===
A music player build with react, react-router,webpack，ES6

实现的功能有音乐播放、上一曲、下一曲、播放进度控制、音量大小控制、播放列表、 歌曲删除、点击播放对应歌曲。

如何运行
# 下载代码
                git clone https://github.com/junhey/React-Music-Player.git
#安装依赖
                npm install
#项目启动
                npm start
#具体可看package.json配置

                npm run dev
#编译打包

                npm run build
